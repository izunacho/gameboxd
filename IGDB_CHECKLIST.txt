STEP 1: Get Twitch Credentials (5 minutes)
─────────────────────────────────────────────────────────────────────────────

1. Go to: https://dev.twitch.tv/console/apps

2. Click "Register Your Application"

3. Fill in:
   ✏️ Application Name: gameboxd
   ✏️ Redirect URL: http://localhost:3000
   ✏️ Category: Application

4. Check terms and click "Create Application"

5. You're now in your app dashboard:

   📋 Copy This:
   ┌─────────────────────────────────────┐
   │ Client ID: (long string)            │ ← Copy this!
   │ Client Secret: [Generate New]       │ ← Click and copy this!
   └─────────────────────────────────────┘

   ⚠️  KEEP THESE SECRET!
      Don't share in email/Slack
      Don't commit to GitHub

STEP 2: Update Your .env.local File (2 minutes)
─────────────────────────────────────────────────────────────────────────────

1. Open file: .env.local (in your gameboxd folder)

2. ADD THIS (or update if exists):

   # IGDB Configuration
   IGDB_CLIENT_ID=paste-your-client-id-here
   IGDB_CLIENT_SECRET=paste-your-client-secret-here

3. Keep your existing settings:
   NEXT_PUBLIC_SUPABASE_URL=...
   NEXT_PUBLIC_SUPABASE_ANON_KEY=...
   SUPABASE_SERVICE_ROLE_KEY=...
   NEXT_PUBLIC_APP_URL=http://localhost:3000

4. REMOVE this line (if present):
   ❌ NEXT_PUBLIC_RAWG_API_KEY=...

STEP 3: Restart Dev Server (1 minute)
─────────────────────────────────────────────────────────────────────────────

In your terminal:

1. Stop current server:
   Press: Ctrl + C

2. Start fresh:
   npm run dev

3. Wait for: "✓ Ready in X.Xs"

⚙️  CREDENTIALS EXPLAINED
────────────────────────────────────────────────────────────────────────────

WHERE THEY GO:
  .env.local (local machine only - not in GitHub)

WHERE THEY STAY:
  Server-side only (Next.js API routes)
  Never exposed in browser
  Never sent to frontend

WHY TWO CREDENTIALS:
  Client ID: Public identifier
  Client Secret: ⚠️  SENSITIVE - keeps it private!

WHAT THEY DO:
  Request access token from Twitch
  ├─ Expires after 1 hour
  ├─ Cached in memory (no re-requests needed)
  └─ Used to call IGDB API

❌ TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────

Problem: "Games not showing"
Solution:
  1. Check .env.local has both IGDB_CLIENT_ID and IGDB_CLIENT_SECRET
  2. Restart server (Ctrl+C, npm run dev)
  3. Check browser console (F12)

Problem: "IGDB_CLIENT_ID must be set"
Solution:
  1. Create .env.local if missing
  2. Add both credentials
  3. Restart server

Problem: "Failed to authenticate"
Solution:
  1. Go to: https://dev.twitch.tv/console/apps
  2. Find your app
  3. Copy Client ID and Secret again carefully
  4. Update .env.local exactly as shown
  5. Restart server

Problem: "Images not loading"
Solution:
  1. Wait a moment (CDN caching)
  2. Refresh browser (F5)
  3. Check browser console (F12)
  4. Try different game
