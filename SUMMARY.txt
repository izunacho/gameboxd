🔑 CREDENTIALS NEEDED

You will need to create 3 FREE accounts:

1. SUPABASE (Database & Auth)
   ├─ Go to: https://supabase.com
   ├─ Create project
   ├─ Get 3 keys to copy:
   │  ├─ Project URL
   │  ├─ Anon Public Key
   │  └─ Service Role Key
   └─ Estimated time: 5 minutes

2. RAWG API (Games Database)
   ├─ Go to: https://rawg.io/api
   ├─ Sign up
   ├─ Get API key
   └─ Estimated time: 2 minutes

3. NODE.JS (Required for Development)
   ├─ Go to: https://nodejs.org
   ├─ Download LTS version
   ├─ Install locally
   └─ Estimated time: 5 minutes

TOTAL SETUP TIME: ~15-20 minutes

📋 QUICK START STEPS

1. Install Node.js (download, install, restart terminal)

3. Get RAWG API key (2 min)
4. Create .env.local file with credentials
